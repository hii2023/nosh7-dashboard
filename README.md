# NOSH7 Dashboard — Subscription Analytics

A web-based analytics dashboard for the **NOSH7 Cloud Kitchen** subscription business — giving the owner and operations team a real-time overview of subscriber metrics and business performance.

## What It Does

Aggregates key NOSH7 subscription data into a single dashboard view — active subscribers, growth trends, renewal rates, and operational KPIs — so the team can make informed decisions fast.

## Key Features

- **Subscriber metrics** — active, new, churned, and renewal-due counts
- **Growth tracking** — subscription growth over time
- **Operational KPIs** — key performance indicators for kitchen and delivery
- **Single-page interface** — everything visible without navigation
- **Built for non-technical users** — clean, readable layout for the business owner

## Tech Stack

- HTML, CSS, Vanilla JavaScript

## Audience

NOSH7 owner (Sapna Ram) and operations team monitoring subscription business health.
